Setup

Three steps

1

Invite the bot

Add RoManage to your Discord server.

2

Verify accounts

Run /verify user:<account> before configuring a server.

3

Link your group

Run /setup group-id:<id> and start using group commands.

Rank-changing commands require the user's verified Roblox account to have the proper group permission.
