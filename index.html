What it does

Built for Roblox group teams

Keep routine Roblox management inside Discord while still checking Roblox-side permissions before sensitive actions run.

01

Account Verification

Verify through RoVer when available, or use RoManager V2's profile challenge system when RoVer is not configured.

02

Multiple Accounts

Save several Roblox accounts, list them, switch the selected account per server, and unverify old accounts.

03

Per-Server Setup

Each Discord server links to its Roblox group with a numeric group ID, keeping configuration clean and explicit.

04

Rank Controls

Promote, demote, and set roles only when the verified Roblox account has the matching group permission.

05

Join Requests

View, accept, and decline Roblox group join requests from Discord with Roblox permission checks in the way.

06

Game Moderation

Use Open Cloud-powered game bans, unbans, and ban logs for connected Roblox experiences.

Permission aware

Discord commands, Roblox authority.

Rank-changing commands require the user to be verified as a Roblox group member with the role management permission.

Server setup is limited to Discord members with Manage Server permission.

Open Cloud keys and Roblox cookies stay in environment variables, never in the public website.

Setup flow

Three clean steps

1

Invite the bot

Add RoManager V2 to your Discord server with the permissions needed for slash commands.

2

Link a group

Run /setup group-id:<id> to bind that Discord server to a Roblox group.

3

Verify accounts

Run /verify, then use account commands like /list-accounts and /switch-account.
