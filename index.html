Commands

The main tools

/verify

Verify a Roblox account through RoVer or a profile challenge.

/list-accounts

See the Roblox accounts saved to your Discord account.

/switch-account

Choose which verified Roblox account is active in a server.

/role

Set a member's Roblox group role if your verified account has permission.

/join-request

View, accept, or decline Roblox group join requests.

/gameban

Ban users from a Roblox experience with Open Cloud support.

Setup

Three steps

1

Invite the bot

Add RoManager V2 to your Discord server.

2

Link your group

Run /setup group-id:<id>.

3

Verify accounts

Run /verify and start using group commands.

Rank-changing commands require the user's verified Roblox account to have the proper group permission.
