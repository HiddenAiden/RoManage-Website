Bot tools

Built around Roblox group work.

Commands are focused on the jobs group staff actually repeat: verifying accounts, changing roles, handling requests, and checking users.

Verify Roblox Accounts

Use RoVer when available, or verify through a profile challenge handled by the bot.

/verify user:username

Save Multiple Accounts

Keep multiple Roblox accounts saved, switch selected accounts, and remove accounts you no longer use.

/switch-account user:username

Manage Ranks

Promote, demote, and set roles only when the verified Roblox account can manage ranks in the group.

/role member:user role:rank

Handle Join Requests

View, accept, or decline pending Roblox group join requests without leaving Discord.

/join-request action:view

Post Announcements

Create Roblox community announcements from a Discord command after the server is linked.

/announce

Moderate Experiences

Use game bans, unbans, and ban logs for Roblox experiences configured with Open Cloud.

/gameban user:username

Roblox-first permissions

No fake authority.

Rank commands check the user's verified Roblox group role before running.

Server setup requires Discord Manage Server permission.

Secrets like Roblox cookies and Open Cloud keys stay out of the website.
