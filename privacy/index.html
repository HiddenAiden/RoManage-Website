Privacy Policy

Last updated: June 26, 2026

Information We Collect

RoManage stores the minimum data needed to provide Discord-to-Roblox verification and group management features. This may include Discord user IDs, Discord server IDs, Roblox user IDs, selected verified accounts, linked Roblox group IDs, command settings, and permission override settings.

Roblox OAuth

When you verify through Roblox OAuth, Roblox sends RoManage a temporary authorization code. RoManage exchanges that code through its Cloudflare Worker to identify the Roblox account that authorized the app. RoManage stores the resulting Roblox account ID and username for verification. RoManage does not store your Roblox password.

How Data Is Used

Stored data is used to verify account ownership, choose which Roblox account is active in a Discord server, enforce Roblox group permission checks, and run configured bot commands.

Data Sharing

RoManage does not sell personal data. Data may be processed by services required to operate the bot, including Discord, Roblox, Cloudflare, GitHub Pages, and the configured database host.

Data Removal

You can remove saved Roblox verification data by using the bot's /unverify command. Server configuration data can be removed by a server administrator or bot operator.

Contact

For privacy questions, contact the RoManage owner through the Discord server where the bot is installed or through the GitHub repository owner.
